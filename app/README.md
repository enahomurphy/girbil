# Girbil
Desktop app for girbil. this is built using framework 7 and codova electron it utilizes

### TECHNOLOGY
- graphql
- [xstate](https://xstate.js.org/)
- [framework7](https://framework7.io/docs/)
- react
- [apollo graphql](https://www.apollographql.com/docs/react/get-started/)

### RUN REQUIREMENTS
you need your env keys setup.
- Run `cp .env.example .env`
- update add your dev keys

### RUN
- Run `yarn install`
- RUn `yarn cordova-electron`
